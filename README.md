# QuadraLearn

**QuadraLearn** is a web-based interactive learning media designed for teaching quadratic functions in grade X using the tutorial learning method
(*Media Pembelajaran Interaktif Berbasis Web Pada Materi Fungsi Kuadrat Kelas X Dengan Model Tutorial*).

---

## 📚 Background

In the field of education, technology plays an important role in improving learning effectiveness and student engagement. In mathematics learning, especially in quadratic function topics, students often face difficulties in understanding concepts such as graphing, determining roots, and identifying characteristics of functions.

Interactive learning media can help overcome these challenges by providing:

* Step-by-step explanations
* Immediate feedback
* Engaging learning experiences

QuadraLearn is developed as a web-based interactive learning media that applies the **tutorial method**, allowing students to learn independently in a structured and guided way.

---

## 🎯 Objectives

* To help students understand quadratic functions more easily
* To provide interactive and structured learning experiences
* To improve student engagement and motivation in learning mathematics

---

## ⚙️ Technical Specifications

* **Backend**: PHP 8.3, Laravel 11
* **Database**: MySQL / SQLite
* **Frontend**: HTML, CSS, JavaScript

---

## 🚀 Get Started

### 1. Clone Repository

```bash
git clone https://github.com/NurFitriYanti02/quadra-sweet.git
cd quadra-sweet
```

---

### 2. Install Dependencies

```bash
composer install
```

---

### 3. Setup Environment

```bash
cp .env.example .env
php artisan key:generate
```

---

### 4. Run Application

```bash
php artisan serve
```

Open in browser:

```
http://localhost:8000
```
